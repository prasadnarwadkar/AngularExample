Registration as a User
You may register as a user anytime. However, your permisions will be limited. Contact system administrator to give you more advanced role.
Permissions
Each page is available to you based on your role. Administrator role has all the permissions.
Support
Please contact support or system administrator here.
User Management
Please use "forgot password" page to reset the password. Valid email is needed to send the reset password link.
Please use "Manage profile" page to update your profile photo (pic). Select image type as .png (preferred) with sizes less than 1 MB.
For doctors
Doctors can use "Manage Schedule" page to manage their schedules and view their appointments.
Doctors can write notes for their appointments while creating or editing the appointments.
Permissions to use the various forms or pages
Granular-level permisisons can be assigned to roles. When users with a certain role logs in, they can only use the pages as per the role permissions. Each page has four permisisons (create, read, update, and delete). These can be set separately for each page and role.
